# Snake Game

- Code for an implementation of the game Snake on a breadboard using a microcontroller
- Components:
  - PIC24FJ64GA002 Microcontroller
  - AQM0802A-RN-GBW LCD Display
  - Adafruit 8x8 LED Matrix Display
  - Sparkfun Joystick
